# 🇵🇱 Witaj w TuxEdu!

**TuxEdu** to otwartoźródłowy projekt, którego celem jest stworzenie kompletnego ekosystemu oprogramowania dla polskiej edukacji.

Tworzymy nowoczesne, bezpieczne i łatwe w użyciu rozwiązania dla szkół, nauczycieli, administratorów oraz uczniów, wykorzystując potencjał systemu Linux oraz oprogramowania Open Source.

Projekt powstaje w oparciu o wieloletnie doświadczenie autora zdobyte podczas pracy jako administrator serwerów, specjalista wsparcia technicznego oraz nauczyciel zawodowych przedmiotów informatycznych.

Każda funkcja powstaje jako odpowiedź na rzeczywiste problemy i potrzeby szkół, nauczycieli oraz szkolnych pracowni komputerowych.

> **TuxEdu nie powstaje w laboratorium. Powstaje w prawdziwej szkolnej pracowni komputerowej.**

# 🇬🇧 Welcome to TuxEdu!

**TuxEdu** is an open-source project focused on building a complete software ecosystem for education.

Our goal is to provide modern, secure and user-friendly solutions for schools, teachers, administrators and students by leveraging Linux and Open Source technologies.

The project is built upon the author's many years of experience as a systems administrator, technical support specialist and vocational computer science teacher.

Every feature is designed to address real-world challenges faced by schools, teachers and computer labs.

> **TuxEdu is not built in a laboratory. It is built in a real school computer lab.**

# 🎯 Project Goals

* 🐧 Build a stable Linux platform for schools.
* 🏫 Simplify administration of school computer labs.
* 👨‍🏫 Support teachers in everyday classroom work.
* 👨‍🎓 Create a friendly environment for students.
* 🔓 Develop fully Open Source software.
* 🤝 Build a community around educational technologies.
